## Building

This is an Android project built with Gradle. To build:

1. Clone the repository
2. Open in Android Studio
3. Build and run

## Requirements

- Android Studio
- Android SDK (minSdk 21, targetSdk 31)
- OpenCV 4.10.0 (included in the repository)
